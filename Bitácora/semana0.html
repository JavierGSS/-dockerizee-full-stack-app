Esta semana aprendí algunas cosas interesantes. Quizá no sea mala idea sub-dividir su presentación en una lista ordenada 😉:

  1. JavaScript
    JavaScript (JS) es el lenguaje de programación más ejecutado en el mundo. Es el "lenguaje de la web". Esto se debe al hecho de que provee un conjunto de tecnologías open source capaces de correr tanto en un navegador como Firefox o Chrome, como en un servidor o en una base de datos. Por tanto, JS le proporciona a lxs desarrolladores un lenguaje unificado para lidiar con el front-end, el back-end y las bases de datos que moviliza una aplicación o página web.

  2. Interfaz de línea de comando
    Fascinante 🤯! La interfaz de línea de comando es una ventana para explorar, administrar y modificar los contenidos de una compu por vía de instrucciones giradas al CPU en una consola, conocidas como prompts. Los prompts deben estar escritos con la ayuda de ciertas combinaciones especiales de símbolos o "comandos" ('cd', 'pwd' o '~/', por ejemplo). Estos comandos nos ayudan a movernos a través de la jerarquía de directorios en los cuales está segmentada la info de la máquina, y también nos permiten realizar ciertas acciones, como copiar, mover o borrar archivos.

  3. GitHub
    GitHub es una plataforma que nos permite llevar un control de versiones de nuestro código. Este tipo de control es de gran utilidad en proyectos que involucran el trabajo conjunto de varias personas, pues nos brinda (entre otros) los recursos para abrir ramificaciones en un código principal que pueden ser editadas sin afectar el trabajo de otrxs desarrolladores o el funcionamiento global del programa. Para este fin se crean repositorios en el sitio web de GitHub, al cual se pueden añadir los archivos y el código que requiere la implementación de la aplicación o página web, y se mantiene un registro riguroso de las versiones anteriores del código, los cambios realizados, el o la responsable de estos cambios y los propósitos que se persiguen con su introducción.

Durante esta semana también se nos pidió instalar algunas apps a nuestra compu, de suerte que ya para el siguiente módulo todxs tengamos a la mano las herramientas básicas del desarrollador web: VS Code, GitHub app, Google Chrome. Se nos expuso brevemente las partes en las que se divide una computadora, así como el sistema en el que se organiza la información en un disco duro (esencialmente, siguiendo una estructura de árbol, en cuya cabeza se encuentra el sistema raíz). También nos fueron expuestas las ventajas que conlleva el uso de Stack Overflow y las diferencias entre el desarrollo front-end y el desarrollo back-end.
